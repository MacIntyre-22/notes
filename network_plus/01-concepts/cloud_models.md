# Cloud Models

```table-of-contents
```
Public
- available to everyone

Private
- own virualized local data center

Hybrid
- mix of public and private

## SaaS
[SaaS (Software as a Service)](network_plus/def.md#SaaS%20(Software%20as%20a%20Service))
- ondemand software
- no loacl install
- provides services with little management
- data is out there

## IaaS
[IaaS (Infrastructure as a Service)](network_plus/def.md#IaaS%20(Infrastructure%20as%20a%20Service))
- aka Hardware as a Service (HaaS)
- outsourcing equipment
- responsible for management and security
- data is out there but with more security control

## PaaS
[PaaS (Platform as a Service)](network_plus/def.md#PaaS%20(Platform%20as%20a%20Service))
- no servers, no software, no maintenance
- someone else handles platform, you handle development
- trained security watch your stuff
- develop your app from whats available on the platform
