## Key Values
- dictionary is made up of objects with key, value pairs
- they can be different types (K, V)

## Unsorted
- put() adds new entries to the end of the array

## Sorted
- add() checks where key should be or if it exists

## Duplicate Keys
- requires generic collection return types for functions like
	- remove
	- get
- can allow the option for specifiying values for these functions
	- would return a single item instead of collection
