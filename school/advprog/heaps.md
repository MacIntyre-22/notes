## What is a Heap?

A heap is a complete binary tree whose nodes contain Comparable objects and are
organized as follows. 

- Each node contains an object that is no smaller (or no larger) than the objects in its descendants
- In a maxheap, the object in a node is greater than or equal to the objects in the node’s descendants
- In a minheap, the relation is less than or equal to
