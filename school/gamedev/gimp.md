## Get Started
- Before you create a new document, set the grid to make it easier to work with. From the menu bar, select Edit ▸ Preferences. 
- Click Default Grid, on the left, and change the Spacing to 32 × 32
- Once you made a new file, select View ▸ Show Grid, then View ▸ Snap to Grid

## Shading

1. In the Layers Palette, click the New Layer button and click OK on the resulting dialog to create a new layer. 
2. Click the tiny Foreground/Background icon in the Tools palette to reset the colors to black and white. 
3. Use the Rectangle Select tool to select the left-most column in the grid, and press Ctrl-, to fill it with black.
4. Use the Rectangle Select tool to select the right-most column in the grid, and press Ctrl-. to fill it with white. 5. 
5. In the Layers palette, change the Mode pop-up to Overlay. 
6. Your left-most column is now nicely shaded, and your right-most column has a nice highlight.
